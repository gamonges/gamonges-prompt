# ドメイン名ブレインストーミング

クリエイティブで覚えやすいドメイン名を提案し、複数の TLD での空き状況を確認する。

## いつ使うか

- 新しいプロジェクトや会社を始めるとき
- 製品やサービスをローンチするとき
- パーソナルブランドを構築するとき
- 希望ドメインが取得済みで代替を探すとき

## 使い方

### 基本

```
リモートチーム向けのプロジェクト管理ツールのドメイン名を提案して。
```

### 好みを指定

```
AI文章作成アシスタントのドメインが欲しい。短くて .ai か .io がいい。
```

### キーワードを指定

```
デザインエージェンシー用に「pixel」か「studio」を使ったドメインを提案して。
```

## 出力例

**ユーザー**: 「開発者向けのコードスニペット共有ツールのドメイン名を提案して」

**出力**:

```
🎯 ドメイン名の提案

## 空いている (.com)
1. ✓ snippetbox.com - 明確で覚えやすい
2. ✓ codeclip.com - 短い（8文字）、素早い共有を連想
3. ✓ devpaste.com - 開発者に馴染みやすく、用途が明確

## 空いている (その他のTLD)
4. ✓ snippet.dev - 開発者ツールにぴったりの .dev
5. ✓ codebox.io - テック感があり、すっきりした名前
6. ✓ sharecode.app - 機能を表す分かりやすい名前

## 取得済み/プレミアム
- codeshare.com (推定 $2,500)
- snippets.com (プレミアム)

## おすすめ: snippet.dev
- 開発者ターゲットに最適
- 短くて覚えやすい
- .dev で信頼感アップ

次のステップ: 取られる前に登録する？もっとバリエーションが欲しい？
```

**参考**: Lenny's Newsletter での Ben Aiad のユースケース

## 良いドメイン名の原則

- **短い**（15 文字以下）
- **覚えやすく**発音しやすい
- **ハイフンなし**（口頭で伝えやすい）
- **ブランド化しやすく**ユニーク

## TLD の選び方

| TLD  | 向いている用途                       |
| ---- | ------------------------------------ |
| .com | 汎用的、信頼性が高い                 |
| .io  | テック系スタートアップ、開発者ツール |
| .dev | 開発者向け製品                       |
| .ai  | AI・機械学習製品                     |
| .app | アプリケーション                     |

## 価格の目安

- 一般的なドメイン: 年間 約$10-15
- プレミアム TLD (.io, .ai): 年間 約$30-50
- 取得済みドメイン: 市場価格
