## スキーマ検証状況

**注意**: 以下のプロパティは `/adr` コマンドテンプレート（`claude/commands/adr.md`）からの推定です。Notion MCP 接続時に `user-Notion:notion-fetch` でデータベースにアクセスし、正確なプロパティ名・型を検証してください。

検証後、このファイルを更新して「推定」の注記を削除してください。

## 採用状況の選択肢（推定）

| 値 | 説明 |
|---|------|
| 提案済み | 検討中のADR |
| 承認済み | 採用が決定したADR |
| 棄却 | 不採用となったADR |
| 非推奨 | 以前は有効だったが現在は推奨しないADR |
| 置き換え済み | 新しいADRで置き換えられたADR |

## 主要プロパティ一覧（推定）

### 識別情報

| プロパティ名 | 型（推定） | 説明 |
|-------------|-----------|------|
| `タイトル` | title | ADRタイトル（【ADR】タイトル【YYYY-MM】形式） |
| `url` | - | ページURL |

### ステータス・分類

| プロパティ名 | 型（推定） | 説明 |
|-------------|-----------|------|
| `採用状況` | select | ADRのステータス（提案済み / 承認済み / 棄却 / 非推奨 / 置き換え済み） |
| `公開` | checkbox | 公開ステータス（Yes / No） |

### メタデータ

| プロパティ名 | 型（推定） | 説明 |
|-------------|-----------|------|
| `Collaborator` | person | 作成者・関係者 |
| `作成日` | date | ADR作成日（YYYY-MM-DD） |
| `最終更新日` | date | 最終更新日（YYYY-MM-DD） |
| `Last edited by` | last_edited_by | 最終編集者 |
| `最終更新から何日経ったか` | formula | 最終更新からの経過日数 |

### コンテンツ（ページ本文）

ADRの本文は Notion ページのコンテンツとして以下のセクションで構成される（プロパティではなくページ内容）:

| セクション | 説明 |
|-----------|------|
| `文脈` | 判断当時の経緯、観点、事情 |
| `決定事項` | 判断の結果として決定した内容 |
| `結果` | この決定に対する評価 |
| `参考` | 参考にしたWebサイトや書籍、議事録のURL |
